A WordPress plugin to make attractive job listings.

== Description ==

Features include:

* Custom post types for available jobs.

== Installation ==

1. Upload `job-board` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I post a new job? =

Log into the WordPress admin area and select "Jobs" from the main navigation and then choose "Add New Job".
